# Vercel Hobby vs Pro: Better Sleep Tonight

## The Showstopper

Vercel's Hobby plan is restricted to **personal, non-commercial use**. A Tempur-Pedic marketing funnel with paid ad traffic is commercial — Vercel can suspend the account at any time. This alone makes Pro mandatory.

## Key Differences

| | Hobby (Free) | Pro ($20/mo) |
|---|---|---|
| **Commercial use** | Prohibited | Allowed |
| **Bandwidth** | 100 GB hard cap | 1 TB + pay-as-you-go |
| **Edge requests** | 1M/month | 10M/month |
| **When you hit limits** | Site goes down, no recovery until next billing cycle | Overages billed, site stays up |
| **Image optimization** | 1,000/month | 5,000/month |
| **Serverless timeout** | 60s | Up to 800s |
| **Speed Insights** | Basic | Full dashboard with filtering |
| **Uptime SLA** | None | None (Enterprise for 99.99%) |

## Why This Matters With $10k/mo Ad Spend

- **Bandwidth risk**: The quiz serves ~2.3MB of video per session plus avatar videos. At scale, 100 GB could be blown through in hours on a good ad day — and the site just goes dark with no way to fix it until the cycle resets.
- **No graceful degradation**: On Hobby, hitting any cap is a hard wall. On Pro, you pay overages and stay live.
- **Monitoring**: Pro's Speed Insights let you track Core Web Vitals and identify conversion-killing performance issues under real ad traffic.

## Bottom Line

At **$20/month**, Pro is a rounding error against $10k in ad spend. It's the minimum viable plan for this use case. Upgrade before the campaign goes live.
