# Sleep Diagnosis Flow - Content Document

This document contains all the text, questions, and responses for the sleep diagnosis conversation. Edit the content below, and the development team will implement the changes.

---

## How This Document Works

The conversation flows like this:
1. **User lands on homepage** (sees intro text and avatar image)
2. **User clicks "Let's Begin"** (avatar video starts)
3. **Avatar speaks** (introduces the topic or asks a question)
4. **User answers** (clicks a button or types something)
5. **Avatar responds** based on what they chose
6. Repeat until we show the recommendation

**Tips for editing:**
- Copy the blank template at the bottom to add new questions
- Edit any text in the "What the avatar says" sections
- For each answer option, write what the avatar should say back
