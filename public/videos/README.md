# Video Assets Directory

Place your encoded video files here:

## Required Files

### Desktop Videos (1920x1080)
- `hero-background.webm` (2-5MB, VP9 codec)
- `hero-background.mp4` (2-5MB, H.264 codec)

### Mobile Videos (1280x720) - Optional but Recommended
- `hero-background-mobile.webm` (1-2MB, VP9 codec)
- `hero-background-mobile.mp4` (1-2MB, H.264 codec)

## Encoding Commands

See CAMPAIGN_IMPLEMENTATION.md for complete ffmpeg encoding instructions.

## Video Guidelines
- Duration: 15-20 seconds for seamless loop
- Content: Peaceful bedroom scene, soft lighting, gentle movements
- No audio tracks (saves file size)
- Test loop point for seamless transition
